# 2026-06-30 当前验收说明

## 这轮新增了什么

- 千人触达不再靠一次性循环，改成持久任务队列：每个客户都有独立状态，支持分批执行、暂停、停止、重启后继续。
- 自动回复新增第一版闭环：从本地微信会话库扫描未读私聊文本，进入回复队列；DeepSeek 判断是否人工接管；普通文本再复用当前受控发送器。
- 朋友圈营销新增安全底座：扫描当前可见朋友圈动态并保存截图，只对白名单候选允许执行互动；非白名单会直接拦截。
- 桌面控制台新增入口：`生成队列`、`跑下一批`、`自动回复`、`朋友圈`。

## 你现在怎么验收

1. 启动软件和服务：运行 `C:\Users\28293\Desktop\Agent\Start-Agent.ps1`。
2. 打开主界面，先确认微信已连接、话术已导入、客户已同步。
3. 在 `发送` 页点击 `生成队列`，看持久队列里待发送人数。
4. 点击 `跑下一批`，每轮默认只跑当前通道允许的小批量，不会一次硬跑 1000 人。
5. 在 `自动回复` 页点击 `扫描新消息`，再点击 `处理下一条`。报价、会员、售后、展厅类应进入人工接管，不应自动发送。
6. 在微信里打开朋友圈可见页面，再到 `朋友圈` 页点击 `扫描可见动态`。只有扫描出的白名单候选才能进入互动。

## 当前边界

- 已验证的是任务系统、队列、自动回复判断、朋友圈白名单门控和截图证据。
- 1000 人稳定的定义是：能生成 1000 条任务、分批跑、失败可追踪、重启可恢复；不会承诺平台风控永不触发。
- 朋友圈真实点赞/评论还没有做按钮级校准和回执校验，当前不会假装成功。
- 自动回复真实发送仍复用私聊触达发送器，所以必须先保证微信窗口校准和一人验证可用。
